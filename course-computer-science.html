    Minimum Eligibility
    • Minimum Age: 21 Years Old
    • Academic Level:
      • At least Bachelor's degree in any discipline; or
      • Matured candidates who are at least 30 years old and above with 8 years of work experience and a Diploma; and
      • Obtained a score of at least 25 marks in the Math section and 20 marks in the Programming section of the admission test.
    • Language Proficiency: Bachelor degree where English is the mode of delivery, or IELTS 6.0, or equivalent.
    Syllabus
    • Teacher–Student Ratio: 1:20
    • Hours of Learning: 600+
    • Hours of Live Sessions: 35+
    • Course Duration: 13 months
    • Content delivered through videos, case studies, projects, and assignments.
    Object oriented programming in Java
    • Understand and implement the basic concepts of Java like variables, datatypes, conditionals, loops, and more.
    • Understand and apply the concepts of Abstraction and Encapsulation in OOPs.
    • Understand and apply the concepts of Inheritance and Polymorphism in OOPs.
    • Learn about basic data structures such as arrays and array lists, perform some array operations, and set up the environment for the upcoming modules.
    Object Oriented Design
    • Understand the importance of Object Oriented Design and UML Diagrams.
    • Learn about unit testing (testing individual units/components of a software), characteristics of Test-Driven Development and Code Refactoring.
    • Learn modern software engineering practices and contribute to an existing software project or code base using version control tools like Git.
    • Design a basic Food Ordering application from scratch using the concepts of SDLC, OOAD, TDD and version control taught in the course.
    Data Structures and Algorithms
    • Learn and implement order of growth, Big-Oh, runtime + memory analysis, and time vs space trade-off. Learn about algorithmic complexity of problems and improve the efficiency of their implementations.
    • Learn about divide-and-conquer techniques such as merge sort and binary search.
    • Learn Stacks & Queues and how to implement them.
    • Understand the usage and application of various important data structures such as Hash tables, Trees, Binary Search Trees, and Heaps. Learn about their features, utility, and solutions to important problems related to these data structures.
    • Learn and implement greedy algorithms and how to use them to solve optimisation problems.
    • Learn about dynamic programming and how to find the best possible solution from among different solutions.
    • Learn various applications and use cases of Graphs; practice essential graph-related questions.
    • Implement DSA interview-focused problems in 12 sessions on concepts like linked lists, stacks and queues, bit manipulation, and more (focused on frequently asked interview problems).
    • Complete an assignment comprising coding questions based on all the above topics.
    User Interfaces & Frontend Development
    • Create basic websites using HTML & CSS.
    • Implement the basics of JavaScript and DOM manipulation to create an interactive website.
    • Learn advanced concepts of JavaScript.
    • Make REST API calls to the backend server and integrate the response accordingly to the front-end.
    • Create a basic frontend for a hotel room booking website based on the concepts learnt so far.
    • Write applications using the React framework and develop professional-grade applications.
    • Implement an interactive Game library application on React.
    • Create the front-end of a movie booking website using HTML, CSS, JavaScript and ReactJS.
    Backend Development
    • Introduction to Relational Database Management Systems and techniques to model relational databases. Use SQL to perform various DML and DDL queries.
    • Introduction to Spring Boot framework and development of a hello-world web application using Spring Boot.
    • Hands-on learning to build data and service layers in your application.
    • Learn about microservices, their use cases, and challenges of microservices-based architecture.
    • Introduction to REST and its intricacies to develop REST APIs.
    • Introduction to Aspect-Oriented Programming; concepts of exception handling and application security.
    • Learn and implement various microservices communication techniques.
    • Apply the concepts learned so far and work on an industry-grade project.
    Software Architecture
    • Learn about distributed systems (user load distributed across various server systems) and techniques to efficiently manage user traffic.
    • Get introduced to design principles, patterns, and styles around which software architectures revolve.
    • Learn how to take high-level design decisions through industry case studies and the factors to consider during the software development lifecycle.
    • Understand the process from idea inception to final deployment; learn DevOps concepts, practices, and principles used in software development lifecycles.
    • Introduction to the Software Development Life Cycle and the steps involved; learn Agile methodologies in detail.
    • Generative AI series for developing software applications and how the field is evolving with generative AI.
    Capstone
    • The capstone project will stitch all the concepts learnt during the program.
